# Inventory Management System

A Python tool that automates inventory tracking for small businesses.

## Features
- Tracks stock levels automatically
- Dashboard showing total products, low stock items, inventory value
- Saves hours of manual spreadsheet work

## How It Works
1. Upload Excel/Google Sheets file
2. Run Python script
3. Get instant dashboard with alerts

## Tech Stack
- Python
- Pandas
- Excel/Google Sheets
